PCars2RoughIdea’s Privacy Policy.What personnel data do we collect?Nothing, this application does not collect any personal information. We do not collect first name, surname name or any part of your name. This application does not collect phone numbers, emails or any personnel contact information.What information do we collect?The application collects and processes your Project Car’s telemetry but only allows the user of the application to export this information (save) to a file so it can be loaded back at a later time. This telemetry data does not contain any personnel information.
